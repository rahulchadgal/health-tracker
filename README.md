# Madhavi Health Tracker

Static GitHub Pages version of the Madhavi Health Tracker.

## Publish on GitHub Pages

1. Create a public GitHub repository named `madhavi-health-tracker`.
2. Upload these files to the repository root:
   - `index.html`
   - `.nojekyll`
   - `README.md`
3. Open repository Settings -> Pages.
4. Set Source to "Deploy from a branch".
5. Set Branch to `main` and folder to `/root`.
6. Save and wait for GitHub Pages to publish.

The app URL will look like:

```text
https://YOUR_USERNAME.github.io/madhavi-health-tracker/
```

## Install on iPhone

1. Open the GitHub Pages URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Name it `Madhavi Health`.
5. Tap Add.

Logs are stored in the browser on each device using `localStorage`.
